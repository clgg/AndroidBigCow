# 未实现与后续事项

## 当前未实现

- 账号系统：暂未实现登录、注册、用户身份体系。
- 云同步：收藏、复习状态、主题设置目前只保存在本地设备。
- 在线题库更新：题库目前来自内置 `assets/question_bank.json`，暂未支持远程拉取或增量更新。
- 导入题库：设置页目前没有实现从文件选择器导入新的题库 JSON/Markdown。
- 导出进度文件：设置页已展示进度 JSON，但暂未接入系统分享、文件保存或剪贴板复制。
- 复习计划算法：目前只有手动状态标记，暂未实现间隔重复、到期时间、每日复习队列。
- 错题机制：目前未区分“错题”和“难题”，只支持收藏、未掌握、已掌握、下次复习。
- 题目编辑：App 内暂不支持新增、编辑、删除题卡。
- 多端适配：已创建 Android/Windows 工程，但 UI 主要按手机竖屏体验实现，未专门优化平板/桌面布局。

## 当前受阻

- Android debug APK 打包：`flutter build apk --debug` 在当前环境中因 Gradle wrapper 下载/文件锁等待超时未完成。`flutter test` 和 `flutter analyze` 已通过，问题更像是本机 Gradle 下载/构建环境阻塞。

## 后续建议优先级

1. 先解决 Gradle 下载/构建问题，产出可安装 APK。
2. 实现进度导出到文件或剪贴板。
3. 增加导入题库功能，支持替换或合并本地题库。
4. 增加间隔复习字段，例如 `nextReviewAt`、`reviewCount`、`lastReviewedAt`。
5. 再考虑账号、云同步和远程题库更新。
