# Android 面试知识点地图

## Java

- 面向对象：封装、继承、多态、接口、抽象类。
- 集合：ArrayList、LinkedList、HashMap、LinkedHashMap、ConcurrentHashMap。
- 泛型：类型擦除、通配符、协变、逆变。
- 并发：线程、锁、volatile、synchronized、AQS、线程池。
- JVM：类加载、内存模型、GC、引用类型。

## Kotlin

- 基础语法：空安全、扩展函数、数据类、密封类、委托。
- 函数式能力：高阶函数、lambda、inline、reified。
- 协程：CoroutineScope、Job、Dispatcher、suspend、Flow。
- Java 互操作：平台类型、SAM、注解、可见性。
- 工程实践：DSL、Kotlin 性能、KTX。

## Android 基础

- 四大组件：Activity、Service、BroadcastReceiver、ContentProvider。
- 生命周期：Activity/Fragment 生命周期、进程优先级、配置变更。
- 资源系统：Resources、Asset、主题、样式、适配。
- Intent：显式、隐式、任务栈、启动模式。
- 权限：运行时权限、后台权限、隐私合规。

## View / 事件 / 绘制

- View 绘制流程：measure、layout、draw。
- 事件分发：dispatchTouchEvent、onInterceptTouchEvent、onTouchEvent。
- 自定义 View：Canvas、Paint、Path、触摸处理。
- 动画：View 动画、属性动画、Transition。
- RecyclerView：复用、缓存、DiffUtil、ItemDecoration。

## Handler / 线程 / 协程

- Handler、Looper、MessageQueue、ThreadLocal。
- HandlerThread、IntentService、WorkManager。
- 线程池参数、任务调度、拒绝策略。
- Kotlin 协程调度、结构化并发、取消。
- 主线程安全、ANR、线程泄漏。

## Jetpack / 架构

- Lifecycle、ViewModel、LiveData、StateFlow。
- Navigation、Room、DataStore、Paging。
- MVVM、MVI、Clean Architecture。
- Compose 状态、重组、副作用。
- 单向数据流、状态恢复、测试。

## 性能优化

- 启动优化：冷启动、懒加载、基线配置文件。
- 内存优化：泄漏、抖动、大图、Bitmap。
- 卡顿优化：Choreographer、Systrace、Perfetto。
- 包体优化：R8、资源压缩、动态特性。
- 稳定性：Crash、ANR、OOM、监控体系。

## Framework / Binder

- Android 系统启动：init、Zygote、SystemServer。
- Binder：驱动、ServiceManager、代理、线程池。
- AMS：进程管理、Activity 启动、任务栈。
- WMS：窗口层级、输入分发、Surface。
- PMS：安装、解析、签名、权限。

## 图形渲染 / OpenGL

- Android 渲染链路：ViewRootImpl、RenderThread、SurfaceFlinger。
- Surface、SurfaceView、TextureView。
- OpenGL ES：上下文、Shader、纹理、FBO。
- EGL：Display、Context、Surface。
- 多媒体渲染：Camera 预览、视频帧、滤镜。

## 网络 / 存储 / 安全

- HTTP、HTTPS、TLS、证书校验。
- OkHttp、Retrofit、拦截器、连接池。
- SQLite、Room、事务、索引。
- DataStore、SharedPreferences、文件存储。
- 加密、签名、防篡改、隐私合规。

## Gradle / 工程化

- Gradle 生命周期、Task、Plugin。
- Android Gradle Plugin、Variant、Flavor。
- 模块化、组件化、依赖管理。
- 构建性能、缓存、并行构建。
- CI、自动化测试、发布流水线。

## 项目实战

- 大型 App 架构设计。
- 启动慢、卡顿、内存泄漏定位。
- 复杂列表、图片加载、离线缓存。
- 登录、支付、推送、WebView。
- 灰度、监控、回滚、兼容性治理。
