# Android 面试刷题题库

这个题库面向 Android 客户端工程师，不硬分初级、中级、高级。简单题可以作为复习，难点通过标签和复习状态标记，方便后续反复温习。

## 使用方式

- 按模块刷题：先选 Java、Kotlin、Android 基础、Framework、OpenGL 等模块。
- 按标签筛选：优先刷 `高频`、`易混`、`底层`、`场景题`。
- 按复习状态推进：把不会或不稳的题标成 `下次复习`。
- 刷题时先口述答案，再对照答案要点补齐细节。

## 题卡字段

- 题目：面试中可能被直接问到的问题。
- 考察点：面试官真正想确认的能力。
- 答案要点：回答时必须覆盖的关键点。
- 深挖追问：用于模拟二面、三面继续追问。
- 常见误区：容易答错或答浅的地方。
- 标签：基础、进阶、高频、底层、易混、场景题等。
- 复习状态：未掌握、已掌握、下次复习。

## 文件结构

- `00-题卡模板.md`：统一题卡格式。
- `01-知识点地图.md`：完整知识点总览。
- `02-Java.md`：Java 语言与并发基础。
- `03-Kotlin.md`：Kotlin 语言、协程和互操作。
- `04-Android基础.md`：四大组件、生命周期、资源和系统能力。
- `05-View事件绘制.md`：View、绘制流程、事件分发和动画。
- `06-Handler线程协程.md`：Handler、线程通信、线程池、协程。
- `07-Jetpack架构.md`：Jetpack、架构设计、状态管理。
- `08-性能优化.md`：启动、内存、卡顿、包体和稳定性。
- `09-FrameworkBinder.md`：Framework、Binder、AMS、WMS、PMS。
- `10-图形渲染OpenGL.md`：渲染管线、Surface、OpenGL ES。
- `11-网络存储安全.md`：网络、持久化、安全与隐私。
- `12-Gradle工程化.md`：Gradle、构建、模块化和 CI。
- `13-项目实战场景题.md`：综合项目设计与排障场景。
